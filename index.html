Technologies

HTML5

  • HTML is used to make the foundations for the front end of your site
  • I use up to date web accessibility principles which will make your site accessible for all
  • The code is written professionally which will ensure your site will be organized and easy to update

CSS

  • CSS is used to style your website
  • Using CSS I can quickly style and update your website so it looks any way you would like.
  • Quickly and easily change fonts, color palettes, layout, sizing, images, videos, etc.

Javascript

  • Javascript(JS) is a programming language that is used to add interactivity to your website
  • Without JS the user would be unable to interact with the site. These hover icons you're using right now were made with JS
  • JS can be used to add almost anything user interaction you'd like to your website

Python

  • Python is a programming language that is used to help create the backend of your site
  • Python is a very powerful language, it is often used for Machine Learning and AI as well as data collection and data processing.
  • Python, in conjuction with a few frameworks and libraries, can be used to add just about any functionality to your website

Django

  • Django is a backend framework to create the functionality of your website
  • Django forms the backend of your website and helps handle security and interactions with the database
  • Using Django we can quickly create/update your site and give it lots of useful features
